# NEXUS — Real-Time Hand Gesture Interface

**NEXUS** is a real-time hand gesture recognition system built with **Python**, **OpenCV**, **MediaPipe**, and **Pygame**.  
It demonstrates live hand tracking, gesture classification, joint angle analysis, and a futuristic cyberpunk HUD interface running at **60 FPS**.

---

## Features

- Live hand landmark tracking with glowing skeleton overlay  
- Gesture classification with confidence scores  
- Joint angle analysis for advanced hand movement detection  
- Cyberpunk HUD interface showing gesture name, hand data, and real-time graphs  
- Neon green/pink/cyan color palette with scanline effect  
- Gesture-controlled particle system reacting to hand movements  
- FPS counter for real-time performance monitoring  

---


## 📋 Installation

  pip install -r requirements.txt
  Run through: python main.py
